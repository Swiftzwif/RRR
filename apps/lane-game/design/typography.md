# Typography System

## Font Families

### Primary Font: Inter
- **Usage**: Body text, UI elements, buttons
- **Fallback**: system-ui, -apple-system, sans-serif
- **Characteristics**: Clean, highly readable, professional

### Display Font: SF Pro Display
- **Usage**: Headings, hero text, important announcements
- **Fallback**: system-ui, -apple-system, sans-serif
- **Characteristics**: Apple's system font, authoritative, modern

## Font Sizes

| Element | Size | Line Height | Usage |
|---------|------|-------------|-------|
| H1 | 40px | 1.2 | Page titles, hero headings |
| H2 | 32px | 1.3 | Section headers |
| H3 | 28px | 1.4 | Subsection headers |
| H4 | 24px | 1.4 | Card titles |
| H5 | 20px | 1.5 | Small headers |
| H6 | 18px | 1.5 | Labels, captions |
| Base | 18px | 1.6 | Body text, paragraphs |
| Small | 16px | 1.5 | Secondary text |
| XS | 14px | 1.4 | Fine print, metadata |

## Font Weights

- **Light (300)**: Rarely used, only for very large display text
- **Normal (400)**: Default for body text
- **Medium (500)**: Emphasized text, buttons
- **Semibold (600)**: Headings, important labels
- **Bold (700)**: Strong emphasis, CTAs

## Usage Rules

### Headings
- Use SF Pro Display for H1-H3
- Use Inter for H4-H6
- Maintain consistent hierarchy
- Never skip heading levels

### Body Text
- Always use Inter for readability
- Maintain generous line height (1.6) for comfort
- Use 18px base size for optimal reading experience

### Interactive Elements
- Buttons: Medium weight (500)
- Links: Normal weight with accent color
- Form labels: Medium weight (500)

### Accessibility
- Minimum 4.5:1 contrast ratio for normal text
- Minimum 3:1 contrast ratio for large text (18px+)
- Respect user's font size preferences
- Support reduced motion preferences
